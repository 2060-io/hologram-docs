# The Hologram app

**Private Messaging Meets Verifiable AI.**

Hologram is an open-source mobile app that gives users a single verifiable user agent for reaching any Hologram verifiable service or AI agent — end-to-end encrypted, credential-authenticated, no account required.

These docs focus on building the **agent side**. The app itself, its features, and the story around it live on [hologram.zone](https://hologram.zone).
